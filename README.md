# mini-game-cluster 🎮

## 🐍 贪吃蛇游戏 (Snake Game)

一款基于 Java Swing 开发的经典贪吃蛇游戏，具有丰富的游戏特性和流畅的操作体验。

### ✨ 游戏特性

- 🎯 **经典玩法**：控制蛇吃掉红色豆子获得分数
- 🌟 **金色豆子**：每25秒出现一次（随进度缩短至10秒），可多个同时存在，吃掉后获得8秒无敌时间，速度翻倍
- 🏛️ **障碍物系统**：吃豆子会生成蓝色障碍物，最多15个，后期生成概率降低
- ⚡ **速度控制**：按住空格加速，按住Shift减速，松开恢复正常速度，无敌状态速度自动翻倍
- 💎 **无敌模式**：吃掉金色豆子后可以摧毁障碍物，蛇身会闪烁，状态栏显示倒计时
- 📊 **计分系统**：综合计算豆子数量、障碍物摧毁数、连击加分和时间惩罚
- 💜 **紫色豆子**：每2分钟出现一次，吃掉后获得清屏道具
- 🩵 **青色豆子**：每3分钟出现一次，吃掉后获得重生标记
- 🎒 **道具系统**：按1/2/3选择道具，按E使用
- ⚠️ **负面BUFF**：分数达到3000分时触发，持续60秒，生成血色墙体
- 🔥 **连击系统**：4秒内连续吃豆或撞障碍物获得连击加分

### 🎮 操作说明

| 按键 | 功能 |
|------|------|
| **W / ↑** | 向上移动 |
| **S / ↓** | 向下移动 |
| **A / ←** | 向左移动 |
| **D / →** | 向右移动 |
| **按住 空格** | 加速模式 |
| **按住 Shift** | 减速模式 |
| **ESC** | 暂停/继续游戏 |
| **R** | 重新开始游戏 |
| **Q** | 返回主菜单 |
| **1/2/3** | 选择道具栏中的道具 |
| **E** | 使用选中的道具 |
| **F5** | 使用重生标记 |
| **F6** | 直接清屏 |

### 📝 计分规则

| 项目 | 分数 |
|------|------|
| 每吃一个豆子 | +100分 |
| 每摧毁一个障碍物 | +300分 |
| 吃豆连击加成 | +20分/连击数 |
| 撞障碍物连击加成 | +100分/连击数 |
| 时间惩罚 | -5分/秒（开局5秒内免扣分） |
| 分数最低为0分 | - |

### 🎒 道具系统

| 道具 | 获取方式 | 效果 |
|------|---------|------|
| 🟣 清屏道具 | 吃掉紫色豆子 | 清除所有障碍物和血色墙体 |
| 🔄 重生标记 | 吃掉青色豆子 | 按F5重生蛇，获得3秒无敌 |

### ⚠️ 负面BUFF规则

- **触发条件**：单局分数达到3000分，一局仅触发一次
- **持续时间**：60秒，分三个阶段
- **阶段一（0-20秒）**：每5秒生成一堵6格横/竖墙
- **阶段二（20-40秒）**：每5秒生成两堵不相连的6格墙（横竖不一）
- **阶段三（40-60秒）**：每5秒生成三个3×3方块墙
- **碰撞规则**：普通状态撞墙死亡，无敌状态撞墙清除墙体并解除无敌

### 🚀 启动方式

#### 方式一：直接运行JAR（推荐分发）
```bash
cd sneck
java -jar SnakeGame.jar
```

#### 方式二：运行启动脚本
直接双击 `start.bat` 文件即可启动

#### 方式三：手动编译运行
```bash
cd sneck
javac -encoding UTF-8 -d bin src/*.java
java -cp bin snakegame.SnakeGame
```

### 📦 分发说明

如果要将游戏分享给朋友：

### 🎨 游戏截图

游戏包含以下视觉元素：
- 🟢 绿色蛇身（无敌时为青色并闪烁）
- 🔴 红色普通豆子
- ⭐ 金色豆子（带橙色光环动画）
- 🔵 蓝色障碍物（摧毁时会闪光）
- 💜 紫色豆子（清屏道具）
- 🩵 青色豆子（重生道具）

### 🔧 游戏机制

1. **反向限制**：蛇只能90°转向，不能直接掉头
2. **动态难度**：每吃5个豆子速度小幅提升
3. **障碍物限制**：最多15个，后期生成概率降低
4. **连击系统**：4秒内连续吃豆或撞障碍物获得连击加分，吃豆显示红色进度条，碰撞显示蓝色进度条
5. **最高分记录**：自动保存到本地文件
6. **暂停同步**：暂停时所有计时一并暂停
