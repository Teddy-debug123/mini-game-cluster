# mini-game-cluster 🎮

## 🐍 贪吃蛇游戏 (Snake Game)

一款基于 Java Swing 开发的经典贪吃蛇游戏，具有丰富的游戏特性和流畅的操作体验。

### ✨ 游戏特性

- 🎯 **经典玩法**：控制蛇吃掉红色豆子获得分数
- 🌟 **金色豆子**：每25秒出现一次（随进度缩短至10秒），吃掉后获得8秒无敌时间
- 🏛️ **障碍物系统**：吃豆子会生成蓝色障碍物，最多15个
- ⚡ **速度控制**：按住空格加速，按住Shift减速，松开恢复正常速度
- 💎 **无敌模式**：吃掉金色豆子后可以摧毁障碍物，蛇身会闪烁
- 📊 **计分系统**：综合计算豆子数量、障碍物摧毁数、连击加分和时间

### 🎮 操作说明

| 按键 | 功能 |
|------|------|
| **W / ↑** | 向上移动 |
| **S / ↓** | 向下移动 |
| **A / ←** | 向左移动 |
| **D / →** | 向右移动 |
| **按住 空格** | 加速模式 |
| **按住 Shift** | 减速模式 |
| **ESC** | 暂停/继续游戏 |
| **R** | 重新开始游戏 |
| **Q** | 返回主菜单 |

### 📝 计分规则

- 每吃一个豆子：+100分
- 每摧毁一个障碍物：+300分
- 连续吃豆连击：+10分/连击
- 时间惩罚：-5分/秒（开局5秒内免扣分）
- 分数最低为0分

### 🚀 启动方式

```bash
cd sneck
javac -encoding UTF-8 -d bin src/*.java
java -cp bin snakegame.SnakeGame
```

或者直接运行 `start.bat` 文件。

### 🎨 游戏截图

游戏包含以下视觉元素：
- 🟢 绿色蛇身（无敌时为青色并闪烁）
- 🔴 红色普通豆子
- ⭐ 金色豆子（带橙色光环动画）
- 🔵 蓝色障碍物（摧毁时会闪光）
- 📊 状态栏显示：得分、豆子数、时间、连击、最高分

### 🔧 游戏机制

1. **反向限制**：蛇只能90°转向，不能直接掉头
2. **动态难度**：每吃5个豆子速度小幅提升
3. **障碍物限制**：最多15个，后期生成概率降低
4. **连击系统**：2秒内连续吃豆获得连击加分
5. **最高分记录**：自动保存到本地文件

### 🛠️ 技术优化

- ✅ 双缓冲绘制（防闪烁）
- ✅ 单定时器主循环
- ✅ 点位生成校验（防止重叠）
- ✅ 暂停时同步暂停所有计时
- ✅ 内存对象复用
