# mini-game-cluster 🎮

## 🐍 贪吃蛇游戏 (Snake Game)

一款基于 Java Swing 开发的经典贪吃蛇游戏，具有丰富的游戏特性和流畅的操作体验。

### ✨ 游戏特性

- 🎯 **经典玩法**：控制蛇吃掉红色豆子获得分数
- 🌟 **金色豆子**：每25秒出现一次，吃掉后获得8秒无敌时间
- 🏛️ **障碍物系统**：每吃一个豆子会在蛇尾生成障碍物
- ⚡ **速度控制**：按住空格加速，按住Shift减速
- 💎 **无敌模式**：吃掉金色豆子后可以摧毁障碍物
- 📊 **计分系统**：综合计算豆子数量、障碍物摧毁数和时间

### 🎮 操作说明

| 按键 | 功能 |
|------|------|
| **W / ↑** | 向上移动 |
| **S / ↓** | 向下移动 |
| **A / ←** | 向左移动 |
| **D / →** | 向右移动 |
| **按住 空格** | 加速模式 |
| **按住 Shift** | 减速模式 |
| **ESC** | 暂停/继续游戏 |

### 🚀 启动方式

```bash
cd sneck
javac -encoding UTF-8 -d bin src/*.java
java -cp bin snakegame.SnakeGame
```

或者直接运行 `start.bat` 文件。

### 📁 项目结构

```
sneck/
├── src/              # 源代码目录
│   ├── GameFrame.java    # 游戏窗口
│   ├── GamePanel.java    # 游戏面板
│   ├── SnakeGame.java    # 游戏主类
│   └── StartPanel.java   # 开始界面
├── bin/              # 编译后的类文件
└── start.bat         # Windows启动脚本
```

### 📝 计分规则

- 每吃一个豆子：+100分
- 每摧毁一个障碍物：+300分
- 每秒扣除：-5分
